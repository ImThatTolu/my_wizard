INSTALLATION
------------

1. Using Drush (https://github.com/drush-ops/drush#readme)

    $ drush en my_wizard

2. Manually

    Enable the module at Administer >> Extend.


CONFIGURATION
-------------
Add a My Wizard block to a block region.
 - Home > Administration > Structure > Block layout
 - Click 'Place block' of the region you want to place a wizard block in.
 - Search for 'My Wizard' in the listed blocks and click 'Place block'.
 - Save the block.
 - Optionally, create another My Wizard block.
